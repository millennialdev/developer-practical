## Interview Instructions

You will be given:

- Client design notes
- Client logo
- A wireframe layout

Your task is to build a website based on the information provided below.

### 1\. Wireframe Implementation

Based on the provided wireframe and client notes, recreate the layout using **Next.js** and **Tailwind CSS**.

- Use the **client notes as the primary source of content**
- Use the **provided client logo**
- You may generate supporting copy using AI if needed, as long as it aligns with the client’s brand and notes
- Focus on clean layout, spacing, hierarchy, and consistency
- Please use **free licensed images only**
- Images can be downloaded from:
     - [https://unsplash.com](https://unsplash.com)
     - [https://freepik.com](https://freepik.com)
     - or other reputable free stock image websites

### 2\. Services Section (Intentional Challenge)

The services section in the wireframe is intentionally left blank.

For this section:

- Design and implement a **services cards section**
- Propose realistic services that fit the client’s business
- Use the client notes as guidance
- You may reference ideas from our portfolio projects:  
  [https://truesocialmarketing.com/website-design/](https://truesocialmarketing.com/website-design/)
- The final result should feel intentional, professional, and well designed

This section is meant to evaluate your design judgment and content structuring skills.

### 3\. Time Limit

You have **4 hours** to complete this challenge.

We understand the time limitation, but we want to evaluate how you work under real world constraints and how much quality you can deliver within that window.

### 4\. Code Submission

- Push your code to a **public GitHub repository**
- Share the repository link with us when finished
- Deployment is **not required**

### 5\. What We Care About

We care about:

- Design quality
- Layout and spacing
- Visual hierarchy
- Responsiveness
- Clean component structure
- Overall polish
